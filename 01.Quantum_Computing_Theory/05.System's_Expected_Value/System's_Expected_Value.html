Estudo do Valor Esperado de Um circuito em duas perspectivas

  • standart basis measurement
  • utilizando um observvel

Todas as medidas de um circuito apenas so realizadas medindo-se a projeo do estado quntico no Eixo Z. Base Sigma Pauli Z. Esta chamada Standard Basis Measurement.

Exemplo:

In[12]:
from qiskit import QuantumCircuit
import math
# Create a circuit with a register of three qubits
circ = QuantumCircuit(1)
# H gate on qubit 0, putting this qubit in a superposition of |0> + |1>.
circ.h(0)
circ.ry(math.pi/5, 0)

# A CX (CNOT) gate on control qubit 0 and target qubit 1 generating a Bell state.

# CX (CNOT) gate on control qubit 0 and target qubit 2 resulting in a GHZ state.

# Draw the circuit
circ.draw('mpl')
Out[12]:

Para este circuito obtemos inicializamos os qubits em 0 e obtemos o seguinte estado quntico:

In[13]:
from qiskit.quantum_info import Statevector
from qiskit.visualization import plot_state_qsphere, plot_bloch_multivector
%matplotlib inline
# get state vector
sv = Statevector.from_instruction(circ).data

# plots
# plot_state_qsphere(sv)
plot_bloch_multivector(sv)
Out[13]:

Medindo o circuito.

Medir o circuito significa calcular a projeo do vetor estado resultante no eixo Pauli Z e descobrir para este estado quntico qual o valor de alfa e beta, ou mais diretamente qual a probabilidade deste estado obter a medida |0> ou |1>, que so as nicas medidas reais possveis para este circuito. A nica coisa que alteraes de um estado quntico provoca a alterao da probabilidade de obter cada um dos dois estados possveis.
